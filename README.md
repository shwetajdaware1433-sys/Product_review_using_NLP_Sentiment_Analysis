
# 🛍️ Product Review Sentiment Analysis

A Machine Learning-based web application that classifies product reviews into **Positive**, **Neutral**, or **Negative** sentiments. The project uses Natural Language Processing (NLP) techniques for text preprocessing and TF-IDF feature extraction, followed by training and evaluating multiple machine learning models. The final model is deployed using **Streamlit**.

## 📌 Features

- Predicts sentiment of product reviews
- Supports three sentiment classes:
  - 😊 Positive
  - 😐 Neutral
  - ☹️ Negative
- Displays prediction confidence
- Interactive Streamlit web interface
- Fast and easy-to-use application

- Accuracy

## ▶️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/Product-Review-Sentiment-Analysis.git
```

Navigate to the project folder

```bash
cd Product-Review-Sentiment-Analysis
```

Install the required libraries

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
